SISTEMAS DE CODIGO ABIERTO

Un sistema de código abierto, en el contexto de la tecnología de la información, se refiere a un sistema informático cuyo código fuente está disponible para que cualquiera lo estudie, lo modifique y lo distribuya de manera libre.

SISTEMAS DE CODIGO CERRADO

Un sistema de código cerrado, también conocido como software propietario o software de código cerrado, es un tipo de software cuyo código fuente no está disponible públicamente y está protegido por derechos de autor y otras restricciones legales.
